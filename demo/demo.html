evenTutorial

evenTutorial is a jQuery plugin, that creates a tutorial for your website

Version

1.0

Files include

1. include jQuery

<script type="text/javascript" src="http://code.jquery.com/jquery-2.1.3.min.js"></script>

2. include evenTutorial/evenTutorial.css

<link rel="stylesheet" type="text/css" href="evenTutorial/evenTutorial.css" />

3. include evenTutorial/evenTutorial.min.js

<script type="text/javascript" src="evenTutorial/evenTutorial.js"></script>

Basic Usage

            $.evenTutorial({
    steps: [
        {
            selector: $("#step1"),
            tip: "What is \"even\""
        },
        {
            selector: $("#step2"),
            tip: "About this plugin"
        },
        {
            selector: $("#step3"),
            tip: "Current version of this plugin"
        },
        {
            selector: $("#step4"),
            tip: "Don't forget to include necessary files"
        },
        {
            selector: $("#step5"),
            tip: "Basic usage of evenTutorial"
        }
    ]
});

This code will start tutorial on page load for selectors defined in selector property with text defined in tip property.

Properties

1. steps

Array of steps objects with selector and tip

Default: 
[]

2. startOnLoad

Start tutorial on page load

Default: 
true

3. allowSkipping

Allow skipping tutorial (places close button on every step)

Default: 
false

4. prevButtonText

"Previous" button text

Default: 
← Previous

5. nextButtonText

"Next" button text

Default: 
Next →

6. exitButtonText

"Exit" button text

Default: 
× Close tutorial

Callbacks

1. onInit

Called when $.evenTutorial initialized

2. onStart

Called when first step shown

3. onShow

Called when step shown

4. onClose

Called when step closed

5. onExit

Called when exit button clicked

Methods

1. $.evenTutorial.start()

Starts tutorial

2. $.evenTutorial.show( step )

Shows step passed in argumnt step, if not passed - opens step exited

step starts from 0

3. $.evenTutorial.previous()

Goes to previous step

4. $.evenTutorial.next()

Goes to next step

4. $.evenTutorial.close()

Closes tutorial

4. $.evenTutorial.destroy()

Destroys $.evenTutorial
